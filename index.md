# Noblit

Noblit is a local-first writing and preview app for macOS.

## App Store Links

- [Privacy Policy](privacy.md)
- [Support](support.md)

## About

Noblit helps users import local text, edit documents, preview Markdown-style content, and export documents, slides, tables, Mermaid diagrams, and ArchMap diagrams.

The first release is designed to work locally without an account or cloud workspace.
