N Noblit

Privacy Policy

Last updated: 2026-07-05

Noblit is a local-first document workbench for macOS and Windows.

Overview

Noblit lets you import text, edit local documents, and preview them as documents, slides, tables, Mermaid diagrams, and ArchMap diagrams. It works without an account or cloud workspace.

Data Noblit stores

Noblit stores app data locally on your device, in a workspace folder in your home directory:

macOS:    ~/Noblit
Windows:  %USERPROFILE%\Noblit

The workspace may contain your documents, continuously autosaved content, local crash-recovery drafts, library metadata, and any files you explicitly export.

How saving works

Noblit autosaves your work continuously as you type — there is no manual Save step. Library documents are written straight to their file. A separate local crash-recovery draft may also be kept so unsaved work can be recovered if the app closes unexpectedly.

Imported files

When you import a file, Noblit copies it into the Noblit workspace. Noblit does not modify the original imported file in place.

Network use

Noblit does not require an account, cloud sync, analytics, advertising, or telemetry. Noblit does not upload your document content to Noblit servers.

Data sharing

Noblit does not sell personal data and does not share document content with third parties.

Your control

You can delete documents and recovered drafts from Noblit's Library, and remove local files from the workspace folder using Finder (macOS) or File Explorer (Windows).

Contact

For privacy questions, use the Noblit Support page.
