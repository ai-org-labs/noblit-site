Noblit Privacy Policy

Last updated: 2026-07-03

Noblit is a local-first writing and preview app for macOS.

Overview

Noblit lets users import text files, edit local documents, preview Markdown, slides, tables, Mermaid diagrams, and ArchMap diagrams, and export files that the user explicitly chooses to create.

Noblit is designed to work without an account or cloud workspace.

Data Noblit Stores

Noblit stores app data locally on the user's Mac. By default, Noblit uses:

~/Noblit

The workspace may contain imported documents, saved Noblit documents, autosaved working drafts, local draft metadata, and user-created exported files when the user chooses an export location.

Imported Files

When a user imports a file, Noblit copies the file into the Noblit workspace. Noblit does not edit the original imported file in place.

Autosaved Drafts

Noblit may autosave unsaved changes locally so users can recover work from the Library. Autosaved drafts are stored locally in the Noblit workspace.

Network Use

The Noblit MVP does not require an account, cloud sync, analytics, advertising, or telemetry. Noblit does not upload document content to Noblit servers.

Data Sharing

Noblit does not sell personal data. Noblit does not share document content with third parties as part of the MVP.

User Control

Users can delete documents and autosaved drafts from Noblit's Library. Users can also remove local files from the Noblit workspace using Finder.

Contact

For privacy questions, use the Noblit Support page.
