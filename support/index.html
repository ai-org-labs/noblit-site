N Noblit

Support

Noblit is a local-first document workbench for macOS and Windows. Write Markdown-like text and preview it as documents, slides, tables, Mermaid diagrams, and ArchMap diagrams.

Common questions

Where can I download Noblit?

Download the latest build from the Noblit releases page and pick the asset for your platform — .dmg for macOS or .msi for Windows.

Preview builds may be self-signed or not notarized yet, so macOS or Windows can show a security warning during installation.

Go to preview release

Is Noblit in the app stores?

Not yet. Noblit is distributed directly. The first preview builds are for validation; trusted Developer ID notarization and public Windows certificate signing are the next distribution hardening steps.

Do I need to save my work?

No. Noblit autosaves continuously as you type and shows a quiet Saved ✓ in the status bar. There is no Save button. If the app ever closes unexpectedly, unsaved changes can be recovered from the Library's Recovered Drafts.

Where does Noblit store files?

By default, in a workspace folder in your home directory:

macOS:    ~/Noblit
Windows:  %USERPROFILE%\Noblit

Documents live in the documents subfolder, and crash-recovery drafts in .noblit/drafts.

Does Noblit edit imported files directly?

No. Noblit imports a copy into the workspace. The original file stays separate.

How do I present slides?

Switch the preview to Slide, or press Present to open your slides in a separate window — handy for a second screen or projector. The editor and preview stay in sync.

How do I get a file out of Noblit?

Use Export to create a Markdown, HTML, or PDF file at a location you choose. Export is always an explicit action, separate from your live document.

Troubleshooting

  • If import does not work, confirm the file is a readable text file.
  • If export does not appear, choose a writable location in the save dialog.
  • If diagrams do not render, check the Mermaid or ArchMap syntax in the source text.
  • To remove Noblit data, close the app and delete the Noblit workspace folder in your home directory.
